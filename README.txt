== Description ==

GiveXpert is an online fundraising SaaS that helps fundraisers manage their donation and subscription forms.
The Wordpress plugin allow to connect organization's website with their GiveXpert forms to add link, progress bar and counters.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `givexpert` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your GiveXpert API tokens
4. Clear Cache.
5. Add GiveXpert blocks from page and post editor.
